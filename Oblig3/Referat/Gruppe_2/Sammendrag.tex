LibGDX som rammeverk for spillene
\item
Box2D som rammeverk for bevegelser og kollisjoner
\item
JUnit og Mockito som testing
\item
FindBugs som statisk analyse
\item
Solarlint som plugin i eclipse/intellij for å kontrollere kodestil
